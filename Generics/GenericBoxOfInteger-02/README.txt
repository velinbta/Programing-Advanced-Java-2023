2. Generic Box of Integer

Create a generic class Box that can store any type. Override the toString() method to print the type and the value of the stored data in the format "{class full name}: {value}".

Use the class that you've created and test it with the class java.lang.Integer. On the first line, you will get n - the number of strings to read from the console. On the next n lines, you will get the actual strings. For each of them, create a box and call its toString() method to print its data on the console.

Examples:

Input 1:
3
7
123
42

Output 1:
java.lang.Integer: 7
java.lang.Integer: 123
java.lang.Integer: 42

Input 2:
5
12
13
14
15
16

Output 2:
java.lang.Integer: 12
java.lang.Integer: 13
java.lang.Integer: 14
java.lang.Integer: 15
java.lang.Integer: 16
