4. Matching Brackets
We are given an arithmetical expression with brackets. Scan through the string and extract each sub-expression.
Print the result back at the terminal.

Examples:

Input 1:
1 + (2 - (2 + 3) * 4 / (3 + 1)) * 5

Output 1:
(2 + 3)
(3 + 1)
(2 - (2 + 3) * 4 / (3 + 1))

Input 2:
(2 + 3) - (2 + 3)

Output 2:
(2 + 3)
(2 + 3)
