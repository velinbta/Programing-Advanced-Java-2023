2. Sum Bytes
Write a program that reads a text file ("input.txt" from the Resources - Exercises(task directory)) and prints on the console the sum of the ASCII symbols of all characters inside the file.

Examples:

Input:
On January 1 , 1533 , 
Michael Angelo, 
then fifty-seven years old, 
writes
from Florence to 
Tommaso de' Cavalieri, 
a youth of noble Roman family,

Output:
12488
