6. Word Count
Write a program that reads a list of words from the file "words.txt" (from the Resources - Exercises(task directory)) and finds how many times each of the words is contained in another file "text.txt" (from the Resources - Exercises(task directory)). Matching should be case-sensitive.
Write the results in the file "results.txt". Sort the words by frequency in descending order.

Examples:

Input:
of which The

Output:
of - 6
which - 2
The - 1
