5. Average Students Grades
Write a program, which reads the name of a student and their grades and adds them to the student record, then prints grades along with their average grade - ordered the output by the students' names.

Input
On the first line N - the number of students, then on the next, N lines student name with grade.

Examples:

Input 1:
7
Stephan 5.20
Maria 5.50
Stephan 3.20
Maria 2.50
Alex 2.00
Maria 3.46
Alex 3.00

Output 1:
Alex -> 2.00 3.00 (avg: 2.50)
Maria -> 5.50 2.50 3.46 (avg: 3.82)
Stephan -> 5.20 3.20 (avg: 4.20)

Input 2:
4
Alex 4.50
Peter 3.00
Alex 5.00
Peter 3.66

Output 2:
Alex -> 4.50 5.00 (avg: 4.75)
Peter -> 3.00 3.66 (avg: 3.33)

Input 3:
5
George 6.00
George 5.50
George 6.00
Alex 4.40
Peter 3.30
Peter 4.50

Output 3:
Alex -> 4.40 (avg: 4.40)
George -> 6.00 5.50 6.00 (avg: 5.83)
Peter -> 3.30 (avg: 3.30)
