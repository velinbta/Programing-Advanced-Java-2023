3. Periodic Table
You are given an n number of chemical compounds. You need to keep track of all chemical elements used in the compounds and at the end, print all unique ones in ascending order:

Examples:

Input 1:
4
Ce O
Mo O Ce
Ee
Mo

Output 1:
Ce Ee Mo O

Input 2:
3
Ge Ch O Ne
Nb Mo Tc
O Ne

Output 2:
Ch Ge Mo Nb Ne O Tc
