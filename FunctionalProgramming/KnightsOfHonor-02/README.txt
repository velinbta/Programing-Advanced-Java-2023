2. Knights of Honor
Write a program that reads a collection of names as strings from the console and then appends "Sir" in front of every name and prints it back onto the console. Use a Consumer<T>.

Examples:

Input 1:
Peter George Alex Stan

Output 1:
Sir Peter
Sir George 
Sir Alex
Sir Stan

Input 2:
Alex George Peter

Output 2:
Sir Alex
Sir George
Sir Peter
